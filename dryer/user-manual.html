1 · Overview

Dryer is a physics-based chaotic percussion and CV generator for VCV Rack 2. A ball tumbles inside a rotating drum fitted with internal vanes. Every collision fires a trigger pulse, outputs a pitch CV, and sends a MIDI note — with velocity proportional to impact speed.

The physics runs in the rotating reference frame of the drum at 250 Hz, applying gravity, buoyancy, centrifugal force, Coriolis deflection, and air drag. The result is deterministic chaos: the same initial conditions always produce the same rhythm, but small parameter changes cascade into radically different patterns.

Design intent: Dryer is a direct software model of a planned Eurorack hardware module. Panel layout, controls, and signal flow mirror the physical build. VCV Rack is both a testing ground and a creative tool in its own right.

2 · Physics Engine

The simulation runs in the drum's rotating reference frame. At each 4 ms step:

  • Gravity — transformed into the rotating frame via drum angle
  • Buoyancy — Archimedes' principle; significant for the Balloon preset
  • Centrifugal — pushes ball outward proportional to ω²·r
  • Coriolis — deflects the ball perpendicular to its velocity in the rotating frame
  • Air drag — quadratic drag relative to the vane-coupled air velocity field inside the drum
  • Wall friction — regularised Coulomb friction when the ball presses against the drum wall

Collisions with the drum wall and each vane face are resolved with restitution and a 50 ms per-surface debounce to prevent multiple triggers from a single contact event.

4 · Outputs

JackSignalDetail
CLK Clock square wave Frequency = Speed × 4 / 60 Hz. At 15 RPM → 1 Hz; at 30 RPM → 2 Hz. Independent of collisions.
TRG Trigger gate 10 ms gate on any ball collision that passes the debounce filter. Fires for drum wall hits and both faces of each vane.
CV Pitch CV (V/oct) Pitch of the most recent collision, quantised to the active scale. 0V = C4. Changes on each unique surface hit.

5 · Knob Controls

KnobRangeEffect
SPEED 0 – 35 RPM Drum rotation speed. Also sets the CLK output frequency. 0 = drum stopped (ball falls under gravity only).
SIZE 40 – 80 cm Drum diameter. Larger drums allow more ball travel between collisions; smaller drums produce denser, faster rhythms.
NUMBER 1 – 7 (integer) Number of vanes. More vanes = more collision surfaces = more pitch variety and denser triggers.
HEIGHT 10 – 50 % Vane height as a fraction of drum radius. Taller vanes create more chaotic ball trajectories.
Tip: All four knobs snap to integer values. For more dynamic pitch variety, increase NUMBER to 6–7 vanes and HEIGHT to 40–50 %. Smaller SIZE drums also encourage the ball to change surfaces more frequently.

7 · Buttons

MOON — Latching toggle

Switches gravity from Earth (9.81 m/s²) to lunar (1.635 m/s² = 1/6 Earth). Moon gravity produces slower, more floating trajectories and sparser collisions. LED illuminates when active.

LINT — Latching toggle

Lint Trap. Suppresses collision events below 0.15 m/s impact speed. Filters out the gentle rolling and sliding contact that occurs when the ball hugs the drum wall, leaving only sharp, high-energy impacts. Useful for cleaning up sparse patches or tightening rhythmic feel. LED illuminates when active.

SCATTER — Momentary button

Reshuffles the mapping between collision surfaces and musical pitches using a Fisher-Yates shuffle. The set of pitches in use does not change — only which surface plays which pitch. Press once for an instant, non-repeating rearrangement. The green LED flashes briefly to confirm the event fired.

Note: Scatter is also available via MIDI CC 7 (value ≥ 64 → fires on rising edge). This allows scatter to be triggered from a DAW, sequencer, or foot pedal.

8 · Physics Display

The circular display shows a real-time top-down view of the drum interior, rendered using NanoVG in VCV Rack's Darkroom-compatible layer.

  • Drum ring — the outer boundary; dim green
  • Vanes — bright green radial lines; count and length track the NUMBER and HEIGHT knobs
  • Rotation tick — a small dot on the rim showing drum angle
  • Ball — colour indicates type: yellow (tennis), tan (sandbag), blue (balloon)
  • Collision flash — ball brightens to white-yellow on impact; a halo glow pulses outward

The display runs in the GUI thread and reads physics state written by the audio thread. It is Darkroom-mode compatible: the background bezel is always opaque, and all illuminated elements render on layer 1.

10 · MIDI

MIDI Output

On each collision, Dryer sends a MIDI Note On (pitch from active scale, velocity from impact speed × 300, clamped to 127) followed by a Note Off after ~150–320 ms (velocity-scaled). Configure the driver and device via the right-click menu under MIDI Output.

Routing to a DAW or standalone synth: Select Windows MIDI as the driver, then choose a virtual MIDI port such as loopMIDI. Set your DAW or synth to receive on the same port. For routing within VCV Rack, select the Loopback driver, then use a MIDI-CV module on the same Loopback device.

MIDI Input (CC Control)

All four knobs, both buttons, and Scatter can be driven by MIDI CC:

CCTargetBehaviour
CC 1SPEED0–127 → 0–35 RPM (overrides knob)
CC 2SIZE0–127 → 40–80 cm (overrides knob)
CC 3NUMBER0–127 → 1–7 vanes (overrides knob)
CC 4HEIGHT0–127 → 10–50 % (overrides knob)
CC 5MOON≥ 64 → on, < 64 → off
CC 6LINT≥ 64 → on, < 64 → off

CC values for SPEED, SIZE, NUMBER, and HEIGHT override the physical knob until VCV is restarted or the patch is reset. Configure the MIDI input driver and device via the right-click menu.

11 · Patching Ideas

Basic percussion voice

TRG → envelope attack; CV → oscillator V/oct; CLK → clock source for other modules.

MIDI-driven synth (standalone target)

MIDI Output → loopMIDI → Surge XT (or any DAW instrument). Set the synth to Poly mode with fast retrigger. Use Sandbag for dense micro-triggers; Balloon for slow, sparse notes.

Generative melodic line

CV → quantiser V/oct → oscillator. Scatter periodically (manually or via CC 7 from a sequencer) to vary the pitch mapping. Use Octave Shift to place the melody in the right register.

CLK as a rhythmically related clock

CLK outputs Speed × 4 / 60 Hz. At 15 RPM the clock is exactly 1 Hz (60 BPM quarter notes). At 30 RPM it is 2 Hz (120 BPM). Patch CLK into a clock divider to generate subdivisions tied to drum speed.

Lint Trap as a dynamics gate

With many vanes and high speed, Dryer generates a continuous stream of micro-triggers. Enable LINT to gate out the quiet contact events and let only hard impacts through — instantly transforms a rolling texture into a snappy percussive hit.

Moon gravity for slow ambient textures

MOON + Balloon + low SPEED + high HEIGHT: the ball drifts slowly, collisions are rare and gentle. Feed CV into a reverb-heavy pad for slow, unpredictable chord changes.
